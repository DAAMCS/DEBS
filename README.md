# DAAMCS Eljure Brute Software
### Принцип работы
```
- Програма реализована целиком на Python с использованием библиотеки requests.

- Принцип работы заключается в отправке POST-запроса ajaxauthorize с последующей работой с овтетом в формате JSON.

- Программа берёт возможные комбинации на основе файла databes.py, который нужно сгенерировать самому пользователю. 

- В самой программе генерация пока не реализована.
```
#### Цепочка операций
```
1. Программа импортирует пары логин-пароль из файла databes.py
2. В цикле происходят POST-запросы формата https://{школа}.eljur.ru/ajaxauthorize?username='username'&password='password'
3. Ответ обрабатывается и в формате JSON извлекается результат запроса (True при успешной авторизации и False соответственно при неудачной попытке)
4. При правильной паре логин-пароль программа выделяет их в консоли зелёным цветом
```
#### База данных
```
- База данных, как сказано выше распологается в файле database.py
- Данные там хранятся в формате двух словарей, в будущем будет добавлена возможность использовать БД менее строгого формата (txt-файлы и пр.)
- Базы с частытми пар логин-пароль можно брать из интернета.
```
